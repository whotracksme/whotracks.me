Information gathered by the WhoTracks.Me website

We log requests to our website via CloudFront's logging mechanism. These logs, which contain visitors' IP addresses, are anonymised daily with a process which encrypts this IP with a key that is subsequently destroyed. After this step, the original IP addresses can not longer be recovered. The original CloudFront logs are deleted after 24 hours. In summary, access logs containing IP addresses are kept for 24 hours. Anonymised logs which are held longer have no identifiers, and cannot be linked to the requestor IP address.
